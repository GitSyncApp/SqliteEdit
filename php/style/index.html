Alphagrid (pro et contra)

Enhance workflow in the creative process. Enables user to quickly change their mind about proportions in grid measurements. Exploration of new grid systems or analysis of old grid systems Opens new creative ideas in a normally very uninspiring and static process. Can quickly synthesize layout mockups for design related problems. The program is meant to be the first step in a creative process and not a replacement for other static layout composers. Main features: Baseline gridsystem vertical and horisontal Column gridsystem Row grid system
